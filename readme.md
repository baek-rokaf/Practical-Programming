### 6.왜 C#인가요?

- C#은 입문자에게 적합한 주류 언어
  - 깔끔 및 간결
  - 사용자의 실수를 방지
  - 뛰어난 개발환경
  - [docs](https://learn.microsoft.com/en-us/dotnet/csharp/)
- 타 프로그래밍 언어의 미래상을 제공
- C# 활용 다양한 프레임워크 존재
  - .NET(Old) .NET Core(New)
- 개체지향(Object-Oriented Programming) 언어

  - 기계와 가까운 사고
  - 로직을 코드로 표현하는 방법과 상태를 저장하는 방법
- 162강 이후:
  - C# 및 최신 프로그래밍 언어에서 찾을 수 있는 기능
    - Linq
    - Class
    - partial Class
    - property
    - extension method
    - 예외처리
